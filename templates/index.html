Рейтинг мов програмування та IT-професій

IEEE опублікував новий рейтинг мов програмування

Інститут інженерів електротехніки та електроніки IEEE Spectrum нещодавно опублікував свій рейтинг мов програмування за 2020 рік, у якому традиційно лідирує Python.

Список трьох фіналістів:

  1. Python
  2. Java
  3. C

Докладніше про рейтинг можна дізнатися по посиланню.

Складено список найпопулярніших професій у сфері IT

Дослідження проведено на підставі опису понад 300 тисяч ІТ-вакансій у містах-мільйонниках, опублікованих у 2016–2018 роках.

Список професій:

  • Програміст
  • Тестувальник
  • Системний адміністратор
  • Системний архітектор
  • Системний аналітик
